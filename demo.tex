\subsection{Callouts}

\technicadefinecallout{tip}{Tip}{TechnicaAdvice}

\begin{note}
Use note callouts for supporting information that should remain close to the
main explanation without interrupting it.
\end{note}

\begin{tip}
Custom callouts use the same visual system as built-in callouts; only the
environment name, displayed title, and semantic color change.
\end{tip}

\begin{warning}
Use warnings for risks or mistakes where the reader should slow down before
continuing.
\end{warning}
